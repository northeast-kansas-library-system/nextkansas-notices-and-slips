Transform borrower dates - datetime fields

  • Datetime fields (such as borrower.lastseen) will display in YYYY-MM-DD HH:MM:SS format by default
  • [% borrower.lastseen %] == [% borrower.lastseen %]
    • -----
  • If you add "| $KohaDates" after a Datetime variable, the date will render in the format specified in system preference _dateformat_.
  • [% borrower.lastseen | $KohaDates %] == [% borrower.lastseen | $KohaDates %]
    • -----
  • If you add "| $KohaDates with_hours => 1" after a Datetime variable, the date will render in the format specified in system preference _dateformat_ with hours:minutes:seconds added at the end
  • [% borrower.lastseen | $KohaDates with_hours => 1 %] == [% borrower.lastseen | $KohaDates with_hours => 1 %]
